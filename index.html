отправляем данные из локального

  • git clone # клонируем репозитоий на свой компьютер
  • git add . # делаем все новые файлы в этой папке отслеживаемыми и готовыми к коммиту (если нужно добавить только один файл, то вместо точки пишем его название)
  • git commit -m "название_коммита" # создаем коммит с описанием изменений
  • git push # отправляем данные из локального репозитория в удаленный (на github) # снова вносим какие-то изменения # возвращаемся к шагу с git add . и проходим цикл заново
  • git pull # получить изменения с сервера и смержить их со своим кодом
